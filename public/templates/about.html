About This Project

Welcome to my very first full-stack project. This project was born from a desire to see how to connect a front-end framework to a homemade back-end. The skills either learned or sharpened while creating this project include:

  • Using Node.js to create a CRUD application.
  • Using SQL databases to store user information
  • Using JWT to authenticate a user
  • Using local storage to store user login state and JWT
  • Angular routing
  • Accessing and modifying http headers
  • CSS animation
  • CSS
  • And much more!
